Critical Area Analysis (CAA) & Morphological Dilation

1. What is Critical Area Analysis (CAA)?

Critical Area Analysis (CAA) is a yield modeling technique used in semiconductor manufacturing to quantify the layout area where a defect can cause a circuit failure (short or open).

Instead of only checking spacing rules, CAA calculates the actual area where a particle defect would create an electrical failure.

Yield Model

Y = exp(-D₀ · Ac)
  • D₀ = defect density
  • Ac = critical area
  • Y = yield

Larger critical area results in lower yield.

Main Steps in CAA

  • Import layout (GDSII/OASIS)
  • Select failure type (Short / Open)
  • Sweep defect sizes
  • Apply geometric operations
  • Compute weighted critical area
  • Estimate yield

CAA is typically performed using tools such as Calibre YieldAnalyzer.

2. Morphological Dilation

Morphological Dilation is a geometric operation that expands a polygon outward by a defined distance (typically the defect radius).

A ⊕ B
  • A = original polygon
  • B = structuring element (usually a circle)

In semiconductor layout analysis, dilation is used to simulate the effect of particle defects.

Why Dilation is Used in CAA

Instead of dropping a defect at every possible location, we expand layout polygons by the defect radius. If expanded polygons overlap, that region represents a potential short.

Mathematical Definition

Ar = { x | distance(x, A) ≤ r }

This represents all points within distance r of polygon A.

Related Morphological Operations

  • Dilation – Grow geometry (used for Short CAA)
  • Erosion – Shrink geometry (used for Open CAA)
  • Boolean AND – Extract overlapping regions
  • XOR – Edge difference analysis

In practice, tools perform polygon grow operations followed by Boolean intersections to extract critical regions.
