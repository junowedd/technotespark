# Infineon ESD307-U1-02N Electrical Characteristics 분석 (쉽게 풀어쓴 설명)

## 1. 제품 개요 요약

-   **제품명**: ESD307-U1-02N\
-   **타입**: Uni-directional TVS (Transient Voltage Suppressor)\
-   **정격 동작 전압 (VRWM)**: 10 V\
-   **정전용량 (CL)**: 270 pF (typical)\
-   **패키지**: 0603 (TSNP-2-2)

이 제품은 주로 **USB VBUS (5V\~9V 영역)** 같은 전원 라인 보호용으로
사용됩니다.

------------------------------------------------------------------------

# 2. Electrical Characteristics 핵심 정리

## (1) Reverse Working Voltage (VRWM = 10 V)

-   의미: 정상 동작 시 TVS가 **도통하지 않고 버틸 수 있는 최대 전압**
-   즉, 10V 이하에서는 보호 소자가 거의 동작하지 않음

👉 USB 5V 라인 보호에 적합

------------------------------------------------------------------------

## (2) Breakdown Voltage (VBR = 11.1 \~ 12.1 V @ 1 mA)

-   의미: Avalanche가 시작되는 전압
-   1mA 전류에서 측정

👉 10V까지는 안전\
👉 11\~12V 이상부터 보호 동작 시작

------------------------------------------------------------------------

## (3) Reverse Leakage Current (IR \< 1 nA typ., max 100 nA @ 10V)

-   의미: 정상 상태에서 흐르는 미세 누설 전류
-   매우 작음 → 전력 손실 거의 없음

👉 배터리 시스템에 유리

------------------------------------------------------------------------

## (4) Line Capacitance (CL = 270 pF typ., 350 pF max @ 1 MHz)

-   의미: 신호 라인에 추가되는 커패시턴스
-   비교적 큰 값

👉 고속 데이터 라인에는 부적합\
👉 전원 라인 보호에 적합

------------------------------------------------------------------------

# 3. Clamping 성능 (핵심 보호 능력)

## (1) TLP 조건 (100 ns)

-   ITLP = 16A → VCL typ 17V
-   ITLP = 30A → VCL typ 18V
-   Dynamic Resistance RDYN ≈ 0.05 Ω

👉 전류가 증가해도 전압 증가폭이 작음\
👉 매우 낮은 동적 저항

------------------------------------------------------------------------

## (2) Surge 8/20µs 조건

-   IPP = 34A → VCL typ 24V (max 29V)

👉 자동차/산업용 서지 보호 가능

------------------------------------------------------------------------

## (3) IEC ESD (±30 kV)

데이터시트 파형 기준:

-   8kV positive → peak 약 35V
-   15kV positive → peak 약 64V
-   30ns 이후 안정 전압 약 15V 수준

👉 초기 overshoot 존재\
👉 이후 빠르게 안정화

------------------------------------------------------------------------

# 4. Dynamic Resistance (RDYN = 0.05 Ω)

공식:

RDYN = ΔV / ΔI

값이 매우 작음 →\
전류가 증가해도 전압 상승이 크지 않음 → 보호 성능 우수

------------------------------------------------------------------------

# 5. 쉽게 정리하면

이 TVS는:

-   ✔ 10V 이하 정상 동작
-   ✔ 11\~12V에서 보호 시작
-   ✔ 최대 34A 서지 전류 처리
-   ✔ 낮은 동적 저항 (0.05Ω)
-   ✔ 대신 커패시턴스가 커서 고속 데이터용은 아님

👉 **전원 라인 보호용 TVS**

------------------------------------------------------------------------
